Lesson 1: Hello World!


The first program that is generally written in a new language is called Hello World!, named such because it involves the coder simply getting the program to print out "Hello World!".
The first component of this program is known as the print statement. The print statement takes an argument after it, and outputs it to the console.

>>> print "hi"
hi
>>> print 9
9
>>> print 'Dave the legend'
Dave the legend

The second component is known as a string. This string is the argument after the print statement, and is the text that is outputted to the console.
A string is simply a piece of text that is surrounded by either single quotes 'dave' or double quotes "dave".

Finally, putting all this together, our program to print out "Hello World!" is as follows:

>>> print "Hello World!"
Hello World!

Lesson 2: Basic Types


There are many types involved in python, but the few we shall cover this lesson are int, float, and str.

Objects in python are grouped by their type, integers being grouped as ints, non-integer values (decimals) being grouped as floats, and as noted earlier, pieces of text grouped as strings (str).

>>> type(9) == int
True
>>> type(2.3) == float
True
>>> type("Hello World!") == str
True

Lesson 3: Basic Math Operations


In order to perform simple mathematical operations in python, simply use the operators as one would expect:

>>> 9 + 5
14
>>> 4.5 - 2.1
2.4
>>> 4.5 * 4
18.0
>>> 8/3
2

Notice how in the last example, although 8 divided by 3 is really equivalent to 2.667, it shows 8/3 as being 2.

This is because we are dividing two integers by each other. However, if you replace at least one of the arguments with a float, the result is also a float:

>>> 8.0/3
2.6666666666666665

Lesson 5: Comments


When making a piece of open-source code that is visible to the public, it is always helpful to add comments to lines of code to explain their purpose.

To make a single-line comment, precede the line with a simple hashtag #

>>> 9 + 5 #This is a comment
14
>>> 2.5 * 2.5 #So is this
6.25
>>> print "This is not a comment" #But this is
This is not a comment

To make a multi-line comment, surround the comments with triple quotes """:

>>> """This is a comment
... and it continues
... and continues
... and continues...
... until you close it"""
'This is a comment\nand it continues\nand continues\nand continues...\nuntil you close it'

Although it prints out the string after you close the quotes, when you run the program the line will be ignored as the string is not processed.

Lesson 7: if, elif, else


An if statement consists of two parts: the if, and then the condition (a boolean). If the condition results in True, the section of code nested within the if statement is run. Otherwise, it gets skipped over, and proceeds to the elif or the else, if either one exists. The elif is another if statement, but it is connected to its parent if in that the code will only proceed to the elif if the if statement is not valid. If the elif and if are invalid, and an else statement exists, the code will proceed to the else, which takes no condition.

if expression1:
    proceed to line 1 if the expression1 is true
    proceed to line 2 if the expression1 is true
    ...
elif expression2:
    proceed to line 1 if the expression2 is true
    proceed to line 2 if the expression2 is true
    ...
else:
    proceed to line 2 if expression1 and expression2 are false
    proceed line 2 if expression1 and expression2 are false
    ...
>>> height_in_inches = 65
>>> if height_in_inches < 60:
...     print "You are shorter than 5 feet"
... elif height_in_inches >= 72:
...     print "You are taller than 6 feet"
... else: #60 <= height_in_inches < 72
...     print "You are between 5 and 6 feet tall"
... 
You are between 5 and 6 feet tall
>>>
