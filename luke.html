Lesson 1: Hello World!


The first program that is generally written in a new language is called Hello World!, named such because it involves the coder simply getting the program to print out "Hello World!".
The first component of this program is known as the print statement. The print statement takes an argument after it, and outputs it to the console.

>>> print "hi"
hi
>>> print 9
9
>>> print 'Dave the legend'
Dave the legend

The second component is known as a string. This string is the argument after the print statement, and is the text that is outputted to the console.
A string is simply a piece of text that is surrounded by either single quotes 'dave' or double quotes "dave".



Finally, putting all this together, our program to print out "Hello World!" is as follows:

>>> print "Hello World!"
Hello World!

Lesson 2: Basic Types


There are many types involved in python, but the few we shall cover this lesson are int, float, and str.

Objects in python are grouped by their type, integers being grouped as ints, non-integer values (decimals) being grouped as floats, and as noted earlier, pieces of text grouped as strings (str).

>>> type(9) == int
True
>>> type(2.3) == float
True
>>> type("Hello World!") == str
True
